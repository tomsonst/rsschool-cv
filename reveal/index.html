## Webpack advantages: - helps bring together your resources - monitors changes and reruns tasks - can transpile next generation JavaScript to older JavaScript standard (ES5) using Babel - can transpile CoffeeScript to JavaScript - can convert inline images to data: URI - allows you to use require () for CSS files - can split the output file into multiple files to avoid slow page load
#### Two installation methods: - global installation - local installation
Global installation 
For installation with yarn	
							
								yarn global add webpack webpack-cli
							
						
For installation with npm	
							
								npm i -g webpack webpack-cli
Local installation 
For installation with yarn	
							
								yarn add webpack webpack-cli -D
						
					
For installation with npm	
							
								npm i webpack webpack-cli --save-dev
Webpack settings
#### Procedure for configuring a webpack: - entry point - conversions to be performed - output point
Entry point
Example file structure
							
								index.js
									imports about.js
									imports dashboard.js
										imports graph.js
										imports auth.js
											imports api.js
							
						
Add the path to the main file to webpack.config.js
							
								module.exports = {
									entry: './app/index.js'
								}
Loaders are needed to convert files
svg-loader 
Install
							
								npm i svg-inline-loader -D 
							
						
Add to webpack.config.js
							
								module.exports = {
									entry: './app/index.js',
									module: {
										rules: [
											{ test: /\.svg$/, use: 'svg-inline-loader' }
										]
									}
								}
css-loader 
Install
							
								npm i css-loader -D 
							
						
Add to webpack.config.js
							
								module.exports = {
									entry: './app/index.js',
									module: {
										rules: [
											{ test: /\.svg$/, use: 'svg-inline-loader' },
											{ test: /\.css$/, use: 'css-loader' }
										]
									}
								}
style-loader 
Install
							
								npm i style-loader -D  
							
						
Add to webpack.config.js
							
								module.exports = {
									entry: './app/index.js',
									module: {
										rules: [
											{ test: /\.svg$/, use: 'svg-inline-loader' },
											{ test: /\.css$/, use: [ 'style-loader', 'css-loader' ] }
										]
									}
								}
file-loader 
Install
							
								npm install file-loader --save-dev
							
						
Add to webpack.config.js
							
								module.exports = {
									entry: './app/index.js',
									module: {
										rules: [
											{ test: /\.svg$/, use: 'svg-inline-loader' },
											{ test: /\.css$/, use: [ 'style-loader', 'css-loader' ] }
											{ test: /\.(png|jpe?g|gif)$/i, use ‘file-loader’ }
										]
									}
								}
babel-loader 
Install
							
								npm i babel-loader -D 
							
						
Add to webpack.config.js
							
								module.exports = {
									entry: './app/index.js',
									module: {
										rules: [
											{ test: /\.svg$/, use: 'svg-inline-loader' },
											{ test: /\.css$/, use: [ 'style-loader', 'css-loader' ] }
											{ test: /\.(png|jpe?g|gif)$/i, use ‘file-loader’ }
											{ test: /\.(js)$/, use: 'babel-loader' }
										]
									}
								}
Output point 
Add to webpack.config.js
						
							module.exports = {
								entry: './app/index.js',
								module: {
									rules: [
										{ test: /\.svg$/, use: 'svg-inline-loader' },
										{ test: /\.css$/, use: [ 'style-loader', 'css-loader' ] }
										{ test: /\.(png|jpe?g|gif)$/i, use ‘file-loader’ }
										{ test: /\.(js)$/, use: 'babel-loader' }
									]
									output: {
										path: path.resolve(__dirname, 'dist'),
										filename: 'index_bundle.js'
									}		
								}							
							}
### Webpack process: 1. The webpack gets the entry point located in ./app/index.js 2. It parses the import / require statements and creates a dependency graph 3. The webpack starts building the bundle by converting the code using the appropriate loaders 4. It builds the bundle and places it in dist / index_bundle.js
Launching the webpack 
Add script to package.json
						
							"scripts": {
								"build": "webpack"
							}								
						
					
And execute the command in the terminal
						
							npm run build
